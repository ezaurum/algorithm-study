예제 26 : 팩토리얼의 재귀 해

재귀를 이용해서 n!을 구하는 함수를 작성한다.

프로그램 : 팩토리얼의 재귀 해

        function main() {
          var n;
          for (n=0; n<13; n++) {
            console.log(n + "! = " + factorial(n));
          }
        }
        function factorial (num) {
          if (num == 0) {
            return 1;
          } else {
            return num * factorial (num-1);
          }
        }
        main();

결과

        0! = 1
        1! = 1
        2! = 2
        3! = 6
        4! = 24
        5! = 120
        6! = 720
        7! = 5040
        8! = 40320
        9! = 362880
        10! = 3628800
        11! = 39916800
        12! = 479001600

예제 26-1 : 피보나치 수열

재귀를 이용해서 피보나치 수열을 구한다.

프로그램 : 피보나치 수열(재귀 해)

        function main2() {
          var n;
          for (n=1; n<=20; n++) {
            console.log(n + " : " + fibonacci(n));
          }
        }
        function fibonacci(num) {
          if (num == 1 || num == 2) {
            return 1;
          } else {
            return fibonacci(num-1) + fibonacci(num-2);
          }
        }
        main2();
