1-3. 순위 매김

예제 3 : 간단한 순위 매기기 방법

시험 점수의 점수별 순위를 구한다.

프로그램 : 순위 매김

        var a = [56, 25, 67, 88, 100, 61, 55, 67, 76, 56];
        var rank = new Array;
        
        function main() {
          for (i=0; i<a.length; i++) {
            rank.push(1);
            for (j=0; j<a.length; j++) {
              if (a[j] > a[i]) {
                rank[i]++;
              }
            }
            console.log(a[i] + "점은 a 목록중에 " + rank[i] + "등입니다.");
          }
        }
        main();

결과

        56점은 a 목록중에 7등입니다.
        25점은 a 목록중에 10등입니다.
        67점은 a 목록중에 4등입니다.
        88점은 a 목록중에 2등입니다.
        100점은 a 목록중에 1등입니다.
        61점은 a 목록중에 6등입니다.
        55점은 a 목록중에 9등입니다.
        67점은 a 목록중에 4등입니다.
        76점은 a 목록중에 3등입니다.
        56점은 a 목록중에 7등입니다.

연습문제 3-1 : 예제3을 개선한 순위 매기기

반복횟수를 줄이기 위한 순위 매김 알고리즘을 생각해본다.

프로그램 : 개선된 순위 매김

          var a = [56, 25, 67, 88, 100, 61, 55, 67, 76, 56];
          var max = 100;
          var min = 0;
          var num = 10;
          var rank2 = new Array();
          function main2() {
            for (i=min; i<=max; i++) {
              rank2.push(0);
            }
            for (i=0; i<num; i++) {
              rank2[a[i]]++;
            }
            rank2[max+1] = 1;
            for (i=max; i>=min; i--) {
              rank2[i] = rank2[i] + rank2[i+1]; 
            }
            for (i=0; i<num; i++) {
              console.log(a[i] + "점은 a 목록중에 " + rank2[a[i]+1] + "등입니다.");
            }
          }
          main2();

결과

          56점은 a 목록중에 7등입니다.
          25점은 a 목록중에 10등입니다.
          67점은 a 목록중에 4등입니다.
          88점은 a 목록중에 2등입니다.
          100점은 a 목록중에 1등입니다.
          61점은 a 목록중에 6등입니다.
          55점은 a 목록중에 9등입니다.
          67점은 a 목록중에 4등입니다.
          76점은 a 목록중에 3등입니다.
          56점은 a 목록중에 7등입니다.
이렇게 할 경우 데이터 개수가 n, 데이터 범위가 m인 경우, n+m번의 반속으로 순위를 매길 수 있다.
